## Integration contract
- Web port: :8080
- API proxy: `/api/v1/*` → `API_URL` (default `http://localhost:3000`)
- Wire format: snake_case JSON
- Refresh: httpOnly cookie `leo_refresh` via BFF only
- Access: in-memory via AuthProvider
- CSRF: `GET /api/auth/csrf` → `leo_csrf` cookie; `X-CSRF-Token` on BFF POSTs
- Auth header: `Authorization: Bearer` on non-public API paths

## Success criteria
- [ ] BFF handlers proxy leo-api; `leo_refresh` httpOnly cookie set/cleared server-side only
- [ ] CSRF `leo_csrf` cookie; mutating BFF POSTs require `X-CSRF-Token`
- [ ] AuthProvider holds `access_token` in memory; exposes `setSession`, `clearSession`, `decodeClaims`
- [ ] `lib/api.ts` Bearer on non-public paths; silent refresh once on 401; session-expired overlay → `/login`
- [ ] QueryClientProvider in root layout; `@tanstack/react-query` added
- [ ] `lib/session.ts` token persistence removed; MFA enrollment pending in AuthProvider memory
- [ ] No `access_token`/`refresh_token` in sessionStorage/localStorage
- [ ] `npm run build` and `npm run lint` pass
